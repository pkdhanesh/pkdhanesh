## 📂 Featured Projects

### 🚗 Germany Used Car Price Analysis & Prediction
*Machine Learning | Data Analysis | Python*

Developed a predictive model to estimate used car prices using real-world German market data (46,000+ records). Performed data cleaning, feature engineering, and regression modeling.

**Results:**
- R² Score: 0.93  
- RMSE: €1,474  
- MAE: €1,081  

**Key Insights:**
- Age and mileage significantly reduce price  
- Horsepower positively impacts price  
- Premium brands retain higher value  

**Tech Stack:** Python, Pandas, Scikit-learn, Matplotlib, Seaborn
